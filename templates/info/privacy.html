We believe that security, privacy, and peace of mind is very important.

  1. Schedr only supports HTTPS connections.

    The connection between your computer and Schedr is encrypted.

  2. Import feature is disabled.

    Schedr's import feature allowed users to import their current course schedule from their school's course registration system. While this was a cool feature, it required users to divulge their university username and password in order to work. We never stored this information and never had any incidents regarding data leaks or breaches, but it's not worth the liability, so the feature has been disabled.

  3. Extra documentation added to registration form.

    Schedr isn't affiliated with any universities. Extra documentation was added to the registration page so students know that Schedr does not require students to register using their university credentials.
